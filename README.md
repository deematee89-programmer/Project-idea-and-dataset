## ✨💄Deema Beauty💄✨
### AI Skincare & Makeup Assistant✨✨

## ------------------------------------------------------------------------------

### 💡 Idea Description:
#### “This project aims to predict skin types such as oily, dry, normal, and combination using machine learning techniques based on user-
#### related skin features.
#### The system then connects the predicted skin type with a real Sephora cosmetics dataset to recommend suitable skincare and makeup 
#### products.
#### This project demonstrates how artificial intelligence can be applied in personalized beauty and skincare recommendations.”

## ------------------------------------------------------------------------------

### Dataset Information

#### 1 - Skin Type Dataset
#### Type: Tabular dataset (CSV)
#### Rows: 2000+
#### Target Column: Skin_Type
#### Task: Classification (Machine Learning)


#### 2 - Sephora Dataset

#### Type: Product dataset (CSV)
#### Rows: 8494+
#### Used for: Product recommendation
#### Key columns: product_name, primary_category, brand_name, price_usd

## ------------------------------------------------------------------------------

### 🎯 Problem Type

#### Skin Type Prediction → Classification Problem
#### Product Recommendation → Rule-based filtering (future enhancement)

## ------------------------------------------------------------------------------

## 🧠 Project Goal

#### To build an AI system that understands skin characteristics and recommends suitable skincare products,
#### making beauty routines more personalized and data-driven.
